HelloGithub

发表于 2022-04-07 更新于 2022-04-23 分类于 编程实战 阅读次数： Waline： 本文字数： 415

 相信许多学编程的小伙伴对自己的实战能力不是很满意，但又不知道怎么去提升自己的编程能力，接下来我给大家分享一个优秀的开源月刊HelloGithub。

 HelloGitHub 分享 GitHub 上有趣、入门级的开源项目。每月 28 号以月刊的形式更新发布，内容包括：有趣、入门级的开源项目、开源书籍、实战项目、企业级项目等，让你用很短时间感受到开源的魅力，爱上开源！

 到目前为止，HelloGithub已经更新了72期，每一期都包含好几个开源项目，并且都十分的有趣，有C、C++、Java、Python…我们可以根据自身情况进行选择。当然如果你是大佬的话，你也可以在上面贡献自己的项目🤪
