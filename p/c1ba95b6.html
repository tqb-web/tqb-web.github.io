《HelloGithub》第01期

发表于 2022-04-08 更新于 2022-04-23 分类于 编程实战 阅读次数： Waline： 本文字数： 1.2k

简介

 分享 GitHub 上有趣、入门级的开源项目。

 这是一个面向编程新手、热爱编程、对开源社区感兴趣 人群的月刊，月刊的内容包括：各种编程语言的项目、让生活变得更美好的工具、书籍、学习笔记、教程等，这些开源项目大多都是非常容易上手，而且非常 Cool。主要是希望大家能动手用起来，加入到开源社区中。会编程的可以贡献代码，不会编程的可以反馈使用这些工具中的 Bug，帮着宣传你觉得优秀的项目Star 项目⭐️

 在浏览、参与这些项目的过程中，你将学习到更多编程知识、提高编程技巧、找到编程的乐趣。

 最后 HelloGitHub 这个项目就诞生了。

兴趣是最好的老师，HelloGitHub 就是帮你找到兴趣！

Tips：如果文中的图获取刷不出来，可以点击这里更好的阅读体验。

内容

以下为本期内容

1 CSS项目

1.1 typo.css

 typo.css：中文网页重设与排版，为了一致化浏览器排版效果，制作最适合中文阅读的网页排版。包括桌面和移动平台。

2 JavaScript项目

2.1 ant-ant-

 ant-ant- ：阿里开源的设置企业级的UI语言设计React。文档，样式偏向实现设计和展示效果于中文。

2.2 VeryNginx

 VeryNginx：一个功能强大且对人类友好的Nginx扩展程序。VeryNginx文档😉

2.3 activate-power-mode

 activate-power-mode：输入爆炸效果🎇

3 Python项目

3.1 awesome-python-webapp

 awesome-python-webapp：廖老师的Python入门教程中的实践项目的代码，在线阅读教程。

3.2 Minos

 Minos：一个基于Tornado/MongoDB/Redis 的社区系统。

3.3 Tushare

 tushare：TuShare 是一个免费、开源的 Python 财经数据接口包，TuShare 文档。

3.4 beijing_bus

 beijing_bus：北京实时公交，可以显示查询的公交到达某站还需多长时间。

4 其他

4.1 archey-osx

 archey-osx电脑：计算机中显示Mac OS X的基本信息。

4.2 Learning-SICP

 Learning-SICP：《计算机程序的构造和解释》公开课。

4.3 awesome-python-cn

 awesome-python-cn：awesome-python 中文版。

4.4 github

github：《GitHub漫游指南》该指南能够让新手学会使用网站，确定加入开源社区，更加熟练地使用这个网站。
